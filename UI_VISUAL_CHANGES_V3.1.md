## Summary of Visual Changes

### Version Display
- **Before**: "Version 2.5"
- **After**: "Version 3.1"

- **IE Mode**: Visible as toggle button labeled [IE]
- **CI Mode**: Completely hidden (not just disabled)
- **Behavior**: One-way switch from IE to CI only

- **IE Mode**: Hidden

All changes maintain the existing UI design and color scheme while adding the requested functionality.
